# XUV Irradiation History of GJ 1132 b

Analysis pipeline for constraining the XUV irradiation history of GJ 1132 b using Kepler flare data, TESS observations, and Bayesian inference.

## Pipeline Steps

1. **Kepler FFD Corner** — MCMC inference of flare frequency distribution parameters
2. **FFD Age Comparison** — Compare FFD across stellar ages
3. **TESS Flare Candidates** — Identify flare candidates in TESS lightcurves
4. **TESS Flare Lightcurves** — Plot individual flare lightcurves
5. **Comprehensive FFD Comparison** — Combined Kepler + TESS FFD analysis
6. **XUV Luminosity Distribution** — Derive XUV luminosity probability distribution
7. **Engle Age Distribution** — Stellar age posterior from Engle relation
8. **Maximum Likelihood** — Maximum likelihood estimation via MaxLEV
9. **Bayesian Posteriors** — Full Bayesian posteriors via alabi/dynesty
10. **XUV Evolution** — Time-integrated XUV irradiation with vconverge
11. **Cumulative XUV and Cosmic Shoreline** — Total XUV dose and habitability threshold
12. **Error Source Comparison** — Decompose error contributions
13. **Kepler vs TESS Comparison** — Cross-validate Kepler and TESS FFD fits
